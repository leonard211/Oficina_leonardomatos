# 🐍 Estruturas Condicionais em Python  
Conjunto de exercícios introdutórios sobre **estruturas condicionais** (`if`, `elif`, `else`) desenvolvidos no âmbito do curso Técnico de Gestão e Programação de Sistemas Informáticos (GPSI).

Os exercícios estão divididos em três níveis: **Básico (B)**, **Intermédio (I)** e **Avançado (A)**.

---

## 📘 Exercícios Básicos (B)

### **B1 — Número Positivo ou Negativo**
Programa que:
- Pede um número ao utilizador  
- Indica se é **positivo** ou **negativo**

### **B2 — Maior de Idade**
Programa que:
- Pede a idade  
- Indica se é **maior** ou **menor de idade**  
Regra: idade ≥ 18 → maior de idade

### **B3 — Número Par ou Ímpar**
Programa que:
- Pede um número  
- Indica se é **par** ou **ímpar**

### **B4 — Comparação de Dois Números**
Programa que:
- Pede dois números  
- Indica qual é o **maior**

### **B5 — Password Simples**
Programa que:
- Pede uma password  
- Se for `"python"` → **acesso permitido**  
- Caso contrário → **acesso negado**

---

## 📙 Exercícios Intermédios (I)

### **I1 — Classificação de Nota**
Programa que:
- Pede uma nota (0 a 20)  
- Classifica como:  
  - ≥ 18 → **Excelente**  
  - ≥ 14 → **Bom**  
  - ≥ 10 → **Suficiente**  
  - < 10 → **Reprovado**

### **I2 — Classificação de Idade**
Programa que classifica:
- < 13 → **Criança**  
- 13–17 → **Jovem**  
- 18–64 → **Adulto**  
- ≥ 65 → **Sénior**

### **I3 — Múltiplo de 3 e 5**
Programa que:
- Pede um número  
- Indica se é múltiplo de **3**, de **5**, de **ambos**, ou de **nenhum**

### **I4 — Login com Utilizador e Password**
Sistema com credenciais:
- Utilizador: `admin`  
- Password: `1234`  
Verifica ambos os campos.

### **I5 — Número dentro de intervalo**
Programa que:
- Pede um número  
- Indica se está entre **10 e 20**

---

## 📕 Exercícios Avançados (A)

### **A1 — Sistema Multibanco Simples**
Programa que:
- Define saldo inicial = **1000**  
- Pede valor a levantar  
- Se saldo suficiente → **levantamento autorizado**  
- Caso contrário → **saldo insuficiente**

### **A2 — Maior de Quatro Números**
Programa que:
- Pede **4 números**  
- Mostra o **maior**

### **A3 — Classificação de IMC (Simplificado)**
Fórmula:  


\[
IMC = \frac{peso}{altura \times altura}
\]



Classificação:
- < 18.5 → **Baixo peso**  
- 18.5–24.9 → **Normal**  
- 25–29.9 → **Excesso de peso**  
- ≥ 30 → **Obesidade**

### **A4 — Sistema de Desconto**
Programa que:
- Pede valor da compra  
- Aplica desconto:  
  - ≥ 100 → **10%**  
  - ≥ 50 → **5%**  
  - < 50 → **sem desconto**  
- Mostra o valor final

### **A5 — Ano Bissexto (Simplificado)**
Programa que:
- Pede um ano  
- Verifica se é divisível por **4**  
  - Se sim → **Ano bissexto**  
  - Caso contrário → **Ano normal**
